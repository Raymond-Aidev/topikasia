## 1. 제품 개요

### 1.1 배경
국립국제교육원(NIIED)이 운영하는 TOPIK IBT는 2023년 11월 도입 이후 2026년 기준 17개국·연 6회로 급성장 중이다. 현재 Coursemos(유비온) 플랫폼에 의존하고 있으며, 응시자 증가(2025년 55만 명 역대 최고)와 글로벌 확산에 맞춰 자체 IBT 솔루션 필요성이 대두되었다.

### 1.2 제품 목표
- 공식 TOPIK IBT와 동일한 UX·기능 흐름을 자체 플랫폼으로 구현
- TOPIK I (듣기·읽기)와 TOPIK II (듣기·쓰기·읽기) 전 과정 지원
- **어드민(Admin)이 회원을 직접 생성하고 시험세트를 배정**하는 운영 체계 구현
- **문제 출제 모듈**: 문제은행(Question Bank)에서 유형별 [문제 불러오기]로 문항을 Import하여 시험 구성 (유형 수 기본 15개, 설정 변경 가능)
- 감독관 모니터링, 답안 관리, 성적 처리까지 통합 운영 가능

### 1.3 사용자 유형

| 역할 | 설명 | 주요 기능 |
|------|------|-----------|
| **응시자** | 어드민이 등록한 수험생 계정 | 로그인 → 배정된 시험 확인 → 응시 |
| **감독관** | 시험 진행 중 실시간 감독 | 응시자 모니터링, 시험 시작/종료 제어 |
| **어드민** | 시스템 전체 운영 관리자 | 회원 생성, 시험세트 배정, 데이터 모니터링 |
| **출제자** | 문항을 작성하는 콘텐츠 담당자 | 문항 생성·편집, 시험세트 구성, IBT 업로드 |

## 2. 시스템 범위

### 2.1 In Scope (v2.0 구현 대상)

**응시자 클라이언트**
- 로그인 → 배정된 시험세트 선택 → 영역별 시험 → 답안 제출 → 종료

**어드민 시스템**
- 회원(응시자) 계정 생성 (ID + 비밀번호 직접 입력)
- 회원별 시험세트 번호 배정 (1인 1세트)
- 회원 현황 조회 및 관리
- 응시 내역 모니터링
- 시험 운영 통계 대시보드

**문제 출제 모듈**
- 문항 유형별 생성 (5가지 유형)
- 시험세트 구성 (문항 선택·순서 배치)
- 세트 미리보기 (실제 응시 화면 미리보기)
- IBT 시스템 업로드 (세트 활성화)
- 어드민에서 문제 출제 모듈 접속 및 세트 선택

**응시 흐름 (시험세트 연동)**
- 응시자 로그인 시 배정된 1개 시험세트만 표시
- 시험세트 선택 후 시험 시작

### 2.2 Out of Scope (v2.0 이후)

- AI 원격 프록터링 (카메라·얼굴인식)
- AI 자동 채점 (서술형)
- 말하기 평가 (Speaking IBT)
- 결제·환불 연동 (응시자 자가 접수 시스템)
- 홈테스트 (무감독 원격 응시)

## 4. 비기능 요구사항

### 4.1 성능
- 동시 접속자 5,000명 기준 응답 시간 3초 이내
- 오디오 스트리밍 지연 500ms 이하
- 답안 자동 저장 주기: 10초마다 서버 동기화
- 어드민 대시보드 데이터 갱신: 30초 주기 자동 갱신

### 4.2 보안
- HTTPS 전 구간 암호화
- 세션 토큰 기반 인증 (JWT, 역할별 분리)
- 어드민/출제자/응시자 권한 완전 분리 (RBAC)
- 답안 데이터 암호화 저장

### 4.3 가용성
- 시험 시간대 99.9% 업타임 보장
- 장애 발생 시 5분 이내 복구 목표 (RTO)

### 4.4 호환성
- 응시자 클라이언트: Chrome 최신, Edge, Safari (1280×720 이상)
- 어드민·출제자: Chrome, Edge (데스크탑 전용)

## 9. 개발 우선순위 및 단계

### Phase 1 (MVP — 핵심 기능)
- 어드민 로그인 + 회원 생성 (ID·비번·시험세트 배정)
- 응시자 로그인 → 배정 시험세트 표시 → 시험 시작
- 듣기·쓰기·읽기 영역 기본 문항 유형 (MCQ, 단답, 서술, 드롭다운)
- 답안 자동 저장 + 제출
- 문제 출제 모듈: 문제은행 유형별 Import + 시험세트 구성 + IBT 업로드

### Phase 2 (기능 완성)
- 읽기 고급 유형 (문장 배열 D&D, 삽입 위치)
- 어드민 대시보드 + 응시 내역 모니터링
- 감독관 실시간 제어 (WebSocket)
- 네트워크 오류 대응 (IndexedDB 임시 저장)
- 시험세트 미리보기

### Phase 2 추가 (기능 완성에 포함)
- 성적 자동 채점 + 공개 시스템 (SCR-S01~S02, SCR-A12~A13)
- LMS 문제 리뷰 + LLM 해설 생성 (SCR-L01~L03)
- 유형별 강점/약점 분석 차트
- **동시시작 시스템**: 시험 예정 일시 설정 → 10초 카운트다운 → 동시 시작 → 늦은 진입 차단 (SCR-W01~W02)

### Phase 2.5 (홈페이지 리디자인)
- 메인 홈페이지 리디자인: GNB + 배너 캐러셀 + 일정 카드 + 퀵 액세스 + 하단 섹션 (SCR-HOME-01)
- 토픽아시아 소개 페이지: 서비스 특장점, 시험 구성, 이용 방법, 응시 규정 (SCR-HOME-02)
- 공용 컴포넌트: GlobalNavigationBar, Footer, BannerCarousel

### Phase 3 (고도화)
- 공지사항 API 연동 (SCR-HOME-03)
- 모바일 반응형 대응
- AI 프록터링 연동
- 서술형 AI 자동 채점 연동
- 엑셀 일괄 등록·내보내기
- 말하기 평가 확장
