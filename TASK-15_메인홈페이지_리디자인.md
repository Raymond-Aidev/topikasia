# TASK-15: 메인 홈페이지 리디자인 (공식 TOPIK 스타일)

> 연관 PRD: HOME-01~08, REG-01~08
> 참고 화면: SCR-HOME-01 ~ SCR-HOME-03
> 연관 문서: [TASK-13 자가접수](./TASK-13_자가접수_응시자화면.md), [PRD](./PRD.md)
> 우선순위: Phase 2.5 (기능 완성 후, 고도화 전)
> 기준 이미지: `공식토픽 메인페이지.png`

---

## 목적 및 범위

현재 단순한 Hero + 카드 리스트인 `LandingPage.tsx`를 공식 TOPIK 홈페이지 레이아웃 기반으로 전면 리디자인한다.
"토픽 소개"를 **"토픽아시아 소개"**로 변경하고, 소개 페이지에는 모의시험 플랫폼으로서의 가치를 강조하는 콘텐츠를 넣는다.

주요 변경:
```
[현재] Hero 배너 + 카드 그리드 (단일 페이지)
  ↓
[변경] GNB + 배너 캐러셀 + 시험 일정 카드 + 퀵 액세스 + 공지사항 + 소개 페이지
```

### 2-1. GlobalNavigationBar (GNB) — 신규 컴포넌트

**파일**: `client/src/shared/components/GlobalNavigationBar.tsx`
**위치**: 모든 공개 페이지 최상단

#### 유틸리티 바 (상단 40px)

```
                                    [로그인] [회원가입] [🌐 한국어 ▾] [🔍 통합검색] [☰ 전체메뉴]
```

| 요소 | 동작 | 조건 |
|------|------|------|
| 로그인 | → `/registration/login` | 비로그인 시 |
| 회원가입 | → `/registration/signup` | 비로그인 시 |
| {이름}님 / 로그아웃 | 로그아웃 처리 | 로그인 시 |
| 🌐 한국어 ▾ | 언어 드롭다운 (향후) | 항상 |
| 🔍 통합검색 | 검색 모달 (향후) | 항상 |
| ☰ 전체메뉴 | 메가 메뉴 (향후) | 항상 |

**스타일:**
- 배경: `#FFFFFF`
- 텍스트: `#333333`, `font-size: 13px`
- 하단 구분선: `1px solid #E0E0E0`

#### 메인 네비게이션 바 (하단 72px)

```
[TOPIK ASIA 로고]  [시험접수] [성적확인]              [토픽아시아 소개] [시험별 안내] [알림·소통] [학습하기]
```

| 요소 | 타입 | 동작 | 스타일 |
|------|------|------|--------|
| TOPIK ASIA 로고 | 이미지/텍스트 | → `/` | border 박스, `font-weight: 800` |
| 시험접수 | CTA 버튼 | → `/registration/schedules` | `bg: #1976D2`, 흰색, 라운드 pill |
| 성적확인 | CTA 버튼 | → `/exam/score` | `bg: #333333`, 흰색, 라운드 pill |
| 토픽아시아 소개 | 메뉴 링크 | → `/about` | `font-size: 17px`, `font-weight: 600` |
| 시험별 안내 | 메뉴 링크 | → `/registration/schedules` | 동일 |
| 알림·소통 | 메뉴 링크 | → (향후 공지사항) | 동일 |
| 학습하기 | 메뉴 링크 | → `/lms` | 동일 |

**스타일:**
- 높이: `72px`
- 배경: `#FFFFFF`
- 하단: `2px solid #E0E0E0`
- CTA 버튼: `padding: 8px 24px`, `border-radius: 24px`
